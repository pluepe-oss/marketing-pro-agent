# MultiChannel AI Marketer

> AI가 SNS 채널별 마케팅 콘텐츠를 자동 생성합니다

## 소개
교육기관·학원·이러닝 사업자를 위한 AI 기반 멀티채널 마케팅 콘텐츠 자동 생성 웹앱.
하나의 입력으로 인스타그램·카카오·블로그·SMS 등 6개 채널에 최적화된 카피를 즉시 생성합니다.

## 주요 기능
- 📷 6개 채널 동시 콘텐츠 생성 (인스타그램·카카오·블로그·SMS·유튜브·이메일)
- 🌏 한국어·영어·일본어·중국어 4개국어 지원
- 🎓 교육 특화 템플릿 5종
- 🔒 API 키 보안 관리 (sessionStorage 4시간)

## 기술 스택
React 18 + TypeScript + Vite + Tailwind CSS + Anthropic Claude API

## 시작하기
1. 패키지 설치: npm install
2. 개발 서버 실행: npm run dev
3. 브라우저에서 http://localhost:5173 접속
4. Anthropic API 키 입력 후 사용

## 빌드
npm run build
